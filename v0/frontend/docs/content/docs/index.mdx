**Forge** is a modern, extensible web framework for Go applications that provides enterprise-grade features including built-in security, observability, and a powerful plugin system. Designed for high-performance applications with clean architecture and developer experience in mind.

<Callout type="info">
Forge is actively maintained and production-ready. Check our [GitHub repository](https://github.com/xraph/forge) for the latest updates and contribute to the project.
</Callout>

## Architecture Overview

Forge follows clean architecture principles with clear separation of concerns:

<Tabs items={['Core Framework', 'Plugin System']}>
  <Tab value="Core Framework">
    Essential framework components:
    - HTTP router with middleware support
    - Dependency injection container
    - Configuration management
    - Built-in security and observability
    - Service layer architecture
  </Tab>
  <Tab value="Plugin System">
    Extensible plugin architecture:
    - Database plugins (PostgreSQL, MySQL, SQLite)
    - Deployment plugins (Docker, Kubernetes)
    - Framework plugins (Gin, Echo, Fiber)
    - Generator plugins for code generation
  </Tab>
</Tabs>

## Technology Stack

- **Framework**: [Forge](https://github.com/xraph/forge) - Modern Go web framework
- **Database**: PostgreSQL, MySQL, SQLite support with connection pooling
- **Caching**: Redis for session caching and rate limiting
- **Observability**: Built-in metrics, logging, and tracing
- **Security**: Authentication, authorization, and security middleware

## Quick Start

Get started with Forge in minutes:

<Callout type="tip">
Make sure you have Go 1.21+ installed and a supported database running.
</Callout>

```bash
# Install Forge CLI
go install github.com/xraph/forge/cmd/forge@latest

# Create a new project
forge new my-app

# Navigate to your project
cd my-app

# Install dependencies
go mod tidy

# Run the application
forge dev
```

```go
package main

import (
    "log"
    "github.com/xraph/forge"
)

func main() {
    // Create Forge app
    app := forge.New(forge.Config{
        Name:    "My App",
        Version: "1.0.0",
    })
    
    // Add a simple route
    app.GET("/", func(c *forge.Context) error {
        return c.JSON(200, map[string]string{
            "message": "Hello from Forge!",
        })
    })
    
    // Add health check
    app.GET("/health", func(c *forge.Context) error {
        return c.JSON(200, map[string]string{
            "status": "healthy",
        })
    })
    
    // Start server
    log.Println("ðŸš€ Server starting on :8080")
    log.Fatal(app.Listen(":8080"))
}
```
